# Vue 基础入门

## 通过脚手架创建项目

### Vue2 官方脚手架为 @vue/cli

1. 通过 `vue create projectName` 快速搭建一个新项目，或者直接通过 `vue serve` 构建新想法的原型。你也可以通过 `vue ui` 通过一套图形化界面管理你的所有项目。

### Vue3 官方脚手架为 create-vue

使用命令 `npm init vue@latest` 创建项目

搭配的工具

1. 构建工具采用 Vite ；
2. 状态管理工具采用 Pinia；
3. 单元测试工具采用 Vitest；
4. 端到端的测试采用 Cypress
5. IDE 配置推荐采用 vscode + Volar 扩展
