# 开发环境的搭建

## 安装相关插件及其作用

1. 使用 nodemon 监控文件变化，自动重启 node；
2. 使用 cross-env 设置环境变量，兼容 mac linux 和 windows；

   ```js
    // 安装
    npm install nodemon cross-env -D

    // 添加执行脚本
    "dev": "cross-env NODE_ENV=dev nodemon ./index.js"
   ```
