子元素的flex值由三部分组成，flex: [flex-grow] [flex-shrink] [flex-basis]

flex-basis 的取值有：auto(默认值，表示如果子元素有设置width就取宽度，没有则取内容的宽度)、长度值、百分比、content。

flex-grow 弹性系数，用于如果子元素的宽度未超过父元素的宽度，父元素剩余的宽度如何给子元素分配，取值是一个 数值，表示剩余空间的一个比值，默认是0，表示从flex-basis取得尺寸后就不再扩展。

flex-shrink，用于如果子元素的总宽度超过父元素的宽度后，如何收缩各自的宽度。取值也是一个数值，默认是1，表示如果空间不够，所有子元素都会以自己的首选尺寸为基准等比例收缩。计算公式：(子元素的flex-shrink*flex-basis)/(所有子元素的(flex-shrink*flex-basis)之和)，取得的比例基数再*该子元素的flex-basis

flex的分配思路：1. 先检查flex-basis，确定假想的主尺寸；2. 确定实际的主尺寸。如果安装假想的主尺寸把各项排布好之后，容器内还有剩余空间，那么由flex-grow系数决定各自扩展多少；如果容器装不下那么多项，则根据flex-shrink系数决定各项如何收缩。

order 可用于重拍子元素的显示顺序，默认为0，负值为向前拍。不会影响tab的切换顺序。

多行的flexbox布局

flex-wrap 控制是否折行显示，可取值有nowrap(默认值)、wrap、wrap-reverse

flex-grow 在多行/多列布局中的计算是以行为单位的，因此项目的可扩展空间以行的宽度为限。

align-content 对多行容器生效(也就是flex-wrap的取值非nowrap时生效)，其作用与justify-content 对主轴内容排布的作用非常相似。

多行flex布局中没有办法控制特定的某一行，如果某一行扩展的太宽，可通过给每个子元素设置max-width解决。

注意：IE9 及更早版本的IE不支持Flexbox。
